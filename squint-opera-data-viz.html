Squint/Opera By The Numbers

2016 - Ongoing

Every year Squint/Opera Melbourne produces a large amount of data. Squint/Opera by the Numbers began in 2015 as a means to get a grip on the data created in the studio. Two years in, we can compare the size of data on a project by project basis, average file size and the number of files generated. Although the data nebula is more abstract, real world utility and insights have been drawn from it.

The visualisation runs on iOS and Android devices along with a desktop app. At any given time up to 20mil points are rendered on screen.

Client: Squint/Opera

During Squint/Opera's 2016 Christmas party we projected the data nebula live. For the event, we implemented our installation network code so the visualisation could be controlled from any mobile with the app installed at the event.

UI/UX

All of the UI/UX was designed and built in-house. The UI was a key factor in ensuring the project was successful.
